# Industrial Automation Simulations

This repository contains industrial automation and material handling system simulations developed as part of academic coursework.

## Projects

### Conveyor System
- Simulation of a conveyor-based material handling process.
- Demonstrates automated movement of materials in an industrial environment.

### Material Handling System
- Simulation of material transportation and handling operations.
- Designed to understand industrial automation concepts and workflows.

## Tools Used
- Factory I/O / Industrial Automation Simulation Software
- Robotics and Automation Concepts

## Purpose
The objective of these simulations is to gain practical exposure to industrial automation, material handling systems, and manufacturing workflows.
